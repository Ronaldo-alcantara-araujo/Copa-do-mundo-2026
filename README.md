# 🏆 Copa do Mundo FIFA 2026

Aplicativo PWA desenvolvido em HTML5, CSS3 e JavaScript puro para gerenciamento de um bolão completo da Copa do Mundo FIFA 2026.

## ✨ Recursos

- 📱 Aplicativo instalável (PWA)
- 🌐 Funcionamento offline
- 💾 Salvamento automático no navegador
- 📂 Backup e restauração em JSON
- 🌙 Tema escuro moderno
- 🏆 Tabela oficial da Copa do Mundo FIFA 2026
- ⚽ Fase de grupos totalmente editável
- 📊 Classificação automática por:
  - Pontos
  - Saldo de gols
  - Gols marcados
- 🥇 Classificação automática para o mata-mata
- 📝 Mata-mata totalmente editável
- 🏁 Atualização automática do campeão
- 🏳️ Bandeiras das 48 seleções
- 📱 Interface otimizada para celulares
- 🍎 Compatível com Android e iPhone
- 🚀 Instalação como aplicativo na tela inicial

## 📋 Estrutura da Competição

- 48 Seleções
- 12 Grupos
- 72 Jogos da fase de grupos
- 16-avos de Final
- Oitavas de Final
- Quartas de Final
- Semifinais
- Disputa de 3º Lugar
- Grande Final

## 💾 Armazenamento

Todos os dados são salvos automaticamente no navegador utilizando Local Storage, permitindo continuar o bolão exatamente de onde parou.

Também é possível:

- Fazer Backup em JSON
- Restaurar Backup
- Limpar todos os resultados

## 📱 Instalação

### Android
Abra o aplicativo no Google Chrome e toque em **Instalar App** quando a opção estiver disponível.

### iPhone (iOS)
Abra no Safari e toque em:

**Compartilhar → Adicionar à Tela de Início**
